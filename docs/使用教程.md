# 使用教程

启动后访问 [http://127.0.0.1:8787](http://127.0.0.1:8787)，进入"总览"页。

顶部导航栏：**总览 / 策略 / 每日选股 / 复盘 / 知识库**。

---

## 总览

- 4 张大卡片：最新观察池日期、最新复盘日期、策略脚本数、知识库文档数
- 4 张复盘卡片：总样本、触板率、封住率、成功晋级数
- 左侧"最近观察池"，右侧"最近复盘"，点日期直接跳详情
- 底部一排快速入口按钮

## 策略管理 (`/strategy`)

### 历史策略脚本

列出所有 `generate_fengmang_breakout_<日期>.py`：
- 点脚本名查看参数详情（自动提取顶层常量 `ROLE_RULES` / `TRIGGER_RULES` / `EXCLUDED_REVIEW`）
- 行尾"📦 存为版本"会把当前脚本的参数快照保存到 `strategy_versions`

### 调参（不修改原脚本）

进入脚本详情页，下方有"编辑参数（导出为 JSON / 修改 / 应用为新版本）"：
- 直接修改右侧 JSON
- 写好标签和说明
- 点击"💾 存为调参版本"——只存到 `system.db`，原 `.py` 文件不动

### 版本对比

`/strategy/version/<id>/diff` 会把当前版本和"同一脚本下"的上一个版本逐字段对比，差异行打"已变"标签。

### 回测

`/strategy/backtest/run` 基于"复盘台账"对最新一个脚本做事后回测：
- 选取该脚本 `NEXT_DATE` 当日台账行
- 计算样本数、触板率、封住率、平均评分
- 落到 `strategy_backtests` 表

不依赖外部行情，**完全用历史复盘事实**做回放，可重复多次。

## 每日选股 (`/picks`)

### 观察池列表

按日期倒序列出所有 `锋芒爆点_*_打板观察池` 目录，标记 CSV / HTML 是否就绪。

### 观察池详情 (`/picks/<日期>`)

- 顶部：同花顺导入 CSV 的代码列表
- 下方：从 `_分析结论.html` 第一个表格解析出来的候选池（分组、代码、名称、角色、评分、优先级、行业题材、封板、成交/换手、触发条件、放弃条件）
- 每行尾部一个表单：标记 (观察/已建仓/持仓/已放弃) + 批注，保存到 `stock_pick_marks` 表

> 标记和批注**完全保存在本系统**，不会污染旧项目的 CSV/HTML。

### 重跑选股脚本

在 `/picks` 页底"历史选股脚本"区域点"▶ 运行"会调用 `python3 generate_fengmang_breakout_*.py`，输出落到 [/system/logs](http://127.0.0.1:8787/system/logs)。

## 复盘管理 (`/review`)

### 复盘日历

按日期倒序列出 `锋芒爆点_复盘迭代/每日复盘/<YYYYMMDD>/` 目录，点击进入：
- 左侧渲染原 `_复盘报告.md`（旧项目原文）
- 右侧追加复盘笔记（保存到 `review_notes` 表）

### 完整台账 (`/review/ledger`)

渲染 `数据/复盘样本台账.csv`，支持按日期、代码/名称筛选。

### 晋级分层池

四个池子都可直接看：
- `/review/bucket/成功晋级池`
- `/review/bucket/失败晋级池`
- `/review/bucket/失败样本池`
- `/review/bucket/失败上板池`

如果你刚更新了台账 CSV，点 `🔄 重建分层池` 会调用旧项目的 `rebuild_promotion_buckets.py`，结果写回旧项目 `数据/晋级分层/` 目录（这是少数主动写旧项目的场景，且是你授权触发的）。

### 经验库 / 策略迭代日志 / 模板

直接渲染：
- `/review/experience` -> `经验沉淀/锋芒爆点_复盘经验库.md`
- `/review/iteration` -> `策略迭代/锋芒爆点_策略迭代日志.md`
- `/review/template` -> `模板/每日复盘模板.md`

### 我的复盘笔记 (`/review/notes`)

跨日期汇总所有手动追加的笔记，按日期倒序。

## 知识库 (`/kb`)

### 浏览

按四个分类树状展示：
1. `01 解析文本/锋芒爆点`、`01 解析文本/锋芒波段`
2. `02 合并阅读稿`
3. `03 知识库主索引`
4. `04 案例验证`

点击任一文档进入查看页：
- `.md` 用内置 Markdown 渲染器
- `.html` 直接嵌入
- `.csv` 渲染成表格（前 200 行）

### 全文检索

顶部搜索框，命中会显示文档名 / 行号 / 片段。

### 知识笔记

`/kb/notes`：自己写的笔记列表，每条可关联 1 个或多个知识库文档的相对路径，查看该文档时自动在右侧显示关联笔记。

## 运行日志 (`/system/logs`)

最近 30 次脚本调用：选股脚本、`rebuild_promotion_buckets.py` 等，含完整 stdout/stderr/退出码/耗时。

排错首选 → `/system/logs`。
